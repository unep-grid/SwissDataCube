The Swiss Data Cube - Viewer allows visualizing, querying, and accessing products generated by the Swiss Data Cube.
Version: 1.0.0 [03.07.2018]

Features
  • Entierly based on Open Source Software.
  • Based on BootLeaf template. HTML5, Javascript, Bootstrap, Leaflet, and typeahead.js. Open source, MIT licensed, and available on GitHub.
  • Fullscreen mobile-friendly map template with responsive navbar and modal placeholders
  • Fully interoperable using OGC standards (e.g., WMS, WCS) provided by GeoServer.
  • Time-enabled tools: slider, side-by-side.
  • Storymaps

The data provided on this site is for informational and planning purposes only.

Absolutely no accuracy or completeness guarantee is implied or intended. All information on this map is subject to such variations and corrections as might result from a complete title search and/or accurate field survey.

Snow Cover map has been generated using the Snow Observations from Space (SOfS) algorithm. Full Metadata.

Landsat 8 Cloud-free Mosaic has been generated using the Custom Mosaic algorithm and all Landsat 8 observations fro 2016. Full Metadata.
