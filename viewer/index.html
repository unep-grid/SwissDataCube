The Swiss Data Cube - Viewer allows visualizing, querying, and accessing the products produced by the Swiss Data Cube.

Features
  • Based on BootLeaf template. ith Bootstrap, Leaflet, and typeahead.js. Open source, MIT licensed, and available on GitHub.
  • Fullscreen mobile-friendly map template with responsive navbar and modal placeholders
  • Fully interoperable using OGC standards (e.g., WMS, WCS).
  • Time-enabled tools: slider, side-by-side.
